# ほうにゃんスタンプ台帳

特別支援のプリント取り組みを、児童ごとにスタンプで管理するローカルWebアプリです。

## 使い方

1. `index.html` をブラウザで開きます。
2. 設定画面で児童を登録します。
3. ホームで児童を選び、プリント1枚ごとに「スタンプを押す」を押します。
4. 報酬画面で、累計解放やスタンプ交換の状況を確認します。

## 保存

- データはこのPCのブラウザ内に保存されます。
- 別PCへ移すときや念のため残したいときは、設定画面からJSONを書き出してください。
- 復元するときは、同じ設定画面からJSONを読み込みます。

## v1でできること

- 児童ごとのスタンプ追加と取消
- 累計スタンプと使用可能スタンプの分離
- 累計10/20/30個でのスタンプ解放
- 20個でシール、40個で着せ替え券の交換枠
- ほうにゃんホーム表示と音声再生ボタンの準備枠
