# ほうにゃんスタンプ台帳

特別支援のプリント取り組みを、児童ごとにスタンプシートで管理するローカルWebアプリです。

## 使い方

1. `index.html` をブラウザで開きます。
2. 先生ページで児童を登録します。
3. 児童ページで児童を選び、プリント1枚ごとに「スタンプをおす」を押します。
4. 先生ページで履歴の取消、報酬交換、バックアップを管理します。

## 保存

- データはこのPCのブラウザ内に保存されます。
- 別PCへ移すときや念のため残したいときは、先生ページからJSONを書き出してください。
- 復元するときは、同じ先生ページからJSONを読み込みます。

## v1でできること

- 児童ページと先生ページの分離
- 児童ページでのスタンプ追加と20個スタンプシート表示
- 先生ページでの児童追加、編集、削除、スタンプ取消
- 今までに集めたスタンプと使えるシートの分離
- 累計20/40/60個でのスタンプ解放
- 使えるシート1枚単位での報酬交換
- 着せ替え交換枠の使用不可表示
- 紙にスタンプを押すような押印アニメーション
- 20個目で出るほうにゃんのシート完成アニメーション
- 新しいスタンプが使えるようになった時のほうにゃん解放アニメーション
